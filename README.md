## Conventions
1. 본인이 정리한 부분은 본인의 이름 브랜치에 기록한다.
    - 브랜치는 영어(소문자)로 설정

    ```
    main(default)
    choijangwoo
    yukminwoo
    kimjaehyun
    kimhyunjae
    ```

2. 각 주차 **세미나** 부분의 개인 링크를 수정할 때는 main에서 브랜치를 분기하여 readme를 수정 후 PR을 작성한 뒤 분기한 브랜치를 삭제한다.

    - 담당한 아이템의 부분을 다음과 같이 수정한다.
    ```
   [item명 - 이름](readme 링크)
    ```
